# spoof-tunnel

نصب‌کننده‌ی رله‌ی دو-سروری برای عبور از فیلترینگ، با مخفی‌سازی IP سرور خارج پشت Cloudflare CDN.

```
کاربر ایران ──VLESS+XHTTP──> RELAY (اروان) ──XHTTP پشت Cloudflare──> CDN ──> EXIT (هتزنر، x-ui) ──> اینترنت
```

- **EXIT (سرور خارج):** x-ui را خودتان نصب می‌کنید. IP پشت Cloudflare مخفی می‌ماند و بلاک‌نشدنی است.
- **RELAY (سرور ایران):** فقط یک فورواردر سبک. پورت‌های کاربر را به دامین CDN می‌رساند.
- کاربر به **IP اروان** وصل می‌شود (سریع، IP ایرانی تمیز)؛ ترافیک از CDN به x-ui می‌رسد.

## نصب

روی هر سرور:

```
bash <(curl -fsSL https://raw.githubusercontent.com/changecoin938/tunnel-me/main/install.sh)
```

سپس از منو:
- روی هتزنر → گزینه‌ی **EXIT**
- روی اروان → گزینه‌ی **RELAY**

## پیش‌نیاز

- یک دامین اضافه‌شده به Cloudflare (یک دامین برای صدها ساب‌دامین کافی است: `s1.example.com`, `s2...`).
- ساب‌دامین باید Proxy (ابر نارنجی) باشد تا IP هتزنر مخفی بماند.

## مقیاس

برای هر سرور خارج جدید: یک ساب‌دامین جدید + اجرای installer با نقش EXIT.
برای هر رله‌ی جدید: اجرای installer با نقش RELAY و دادن همان دامین CDN.

## مدیریت

منوی گزینه‌ی ۴: ری‌استارت، لاگ زنده، بازسازی کانفیگ، حذف.
سرویس systemd است؛ بعد از قطع برق خودکار بالا می‌آید.
